# Staff Announcement — For Triamudom Family

Google Apps Script email system for sending personalized staff selection announcements for Triamudom Family events.

## What it does

Reads recipient data from a Google Sheet, generates a styled HTML email, sends it via GmailApp, and writes the sending status back to the sheet.

**Current use case:** Staff selection announcement for the ม.4 TU89 parent meeting.

## Tech Stack

- Google Apps Script
- Google Sheets (SpreadsheetApp)
- GmailApp / MailApp
- HTML email with inline CSS (email-client safe)
- Tailwind CSS (preview templates only)
